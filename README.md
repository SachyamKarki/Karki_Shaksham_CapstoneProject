ServiceHub

ServiceHub is a web platform that connects users with trusted service providers around the people who are willing to work for certain time as freelancer.The website offers a variety of home and personal services including carpentry, cleaning, laundry, childcare, gardening, and home repairs.

Features
Browse available service provider who are willing to help you
Contact form for inquiries and support
Responsive design that works on all devices including tablet and mobile ohone
Professional and user-friendly interface
FAQ section for common questions
There is a hover in effect in home page videos
Trigger of alert when mouse positioned in Our Services in home page 
There is also a validation in the form in cotact page. 


Project Structure
index.html - Home page
services.html - Services listing page
contact.html- Contact and support page with form
about.html - About us page
login.html- Login and signup page
styles.css - Main stylesheet
product.html- All the used and home made products


There are also relative Meta Tags addd in each html pages to get better SEO ranking.
